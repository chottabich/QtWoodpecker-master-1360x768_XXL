TOOL DATABASE

The tool database utility enables the user to store additional information about the tools listed in the linuxcnc tool.tbl file. This extra information is kept in a SQLITE database. On startup, the program checks if the database exists and creates one if needed. If the database exists, it is updated with any new information in the tool.tbl file. If the tool.tbl contains tools not in the database, they are added. If the database contains tools not in the tool.tbl, they are deleted.

Usage

  • To view tool offsets, click the OFFSET tab. The TOOLS tab shows additional tool information not stored in the tool.tbl
  • A tool is selected by clicking on the checkbox on the left of the display.
  • Presently, only 1 tool can be selected at a time.
  • A combobox is used to select an icon for the selected tool.
  • To use tool icons, there must be a directory in the CONFIG folder called tool_icons.
  • The tool_icons must be .png images. This enables them to have transparent backgrounds.
  • The preferred icon size is 160 x 270 pixels.
  • The ADD button adds a new tool using the next lowest tool number available.
  • The DELETE button removes the selected tool from the tool.tbl
  • The LOAD button issues a M61 Qn MDI command where n is the selected tool number.
  • The UNLOAD button issues a M61 Q0 MDI command, which tells linuxcnc there is no tool in the spindle.
  • ENABLE EDIT enables editing of individual table cells.
  • EXPORT TABLE creates a PDF file of both the OFFSET table and the TOOL table. The filename is selected via a file chooser dialog.
  • Modified tool parameters will only be saved if the machine is ON and the interpreter is IDLE.

The tool database file in the config directory is called tool_database.db. It is advised to maintain regular backups of this file. If it becomes corrupted, all information the user has entered may be lost and will have to be re-entered in a new database.
